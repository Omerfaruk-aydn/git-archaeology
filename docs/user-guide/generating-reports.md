# Rapor Oluşturma Rehberi

## Rapor Türleri

### 1. Kapsamlı Rapor

Tüm analiz sonuçlarını içeren detaylı rapor.

**İçerik:**
- Proje genel bakışı
- İstatistikler
- Yazar katkıları
- Dosya analizi
- Mimari değerlendirmeler
- Güvenlik notları
- Öneriler

### 2. Özet Rapor

Kısa ve öz rapor.

**İçerik:**
- Temel istatistikler
- Önemli değişiklikler
- Kısa öneriler

### 3. Güvenlik Raporu

Güvenlik odaklı analiz.

**İçerik:**
- Güvenlik açıkları
- Hassas veri sızıntıları
- Güvenlik önerileri
- Uyumluluk durumu

### 4. Mimari Rapor

Mimari analiz.

**İçerik:**
- Mimari yapı
- Bağımlılıklar
- Modülerlik
- Kod kalitesi
- Mimari öneriler

### 5. Legacy Kod Raporu

Legacy kod analizi.

**İçerik:**
- Legacy kod bölgeleri
- Bakım zorlukları
- Modernizasyon önerileri
- Öncelik sıralaması

## Rapor Oluşturma

### Adım 1: Rapor Formunu Açın

Depo detay sayfasında "Rapor Oluştur" butonuna tıklayın.

### Adım 2: Rapor Türünü Seçin

Kullanım amacınıza uygun rapor türünü seçin.

### Adım 3: Ayarları Yapın

| Ayar | Açıklama | Varsayılan |
|------|----------|------------|
| Format | Çıktı formatı | Markdown |
| Dal | Analiz edilecek dal | main |
| Tarih Aralığı | Belirli dönem | Tümü |
| Dosya Yolları | Belirli dosyalar | Tümü |

### Adım 4: Oluşturun

"Rapor Oluştur" butonuna tıklayın.

## Rapor Formatları

### Markdown

- Hafif ve hızlı
- Kolay editör desteği
- Versiyon kontrolü ile uyumlu

### HTML

- Zengin görsellik
- Tarayıcıda görüntüleme
- Yazdırma desteği

### PDF

- Profesyonel görünüm
- Dış paylaşıma uygun
- Arşivleme için ideal

## Rapor Görüntüleme

### Online Görüntüleme

Rapor oluşturulduktan sonra otomatik olarak görüntülenir.

### İndirme

"İndir" butonu ile dosyayı bilgisayarınıza kaydedin.

### Paylaşma

Rapor URL'sini kopyalayarak paylaşın.

## Rapor Yönetimi

### Rapor Listesi

Oluşturulan tüm raporlar "Raporlar" sayfasında listelenir.

### Filtreleme

- Rapor türüne göre
- Tarihe göre
- Format'a göre

### Silme

Gereksiz raporları silebilirsiniz.

## Rapor Özelleştirme

### Özel Dosya Seçimi

Belirli dosyaların analizini içeren raporlar oluşturun.

### Tarih Filtresi

Belirli bir dönemdeki değişiklikleri analiz edin.

### Dal Seçimi

Belirli bir dalın analizini yapın.

## İyi Rapor Oluşturma İpuçları

1. **Doğru türü seçin**: Amacınıza uygun rapor türünü seçin
2. **Tarih aralığını daraltın**: İlgili dönemleri analiz edin
3. **Dosya filtresi kullanın**: Önemli dosyalara odaklanın
4. **Format seçimi**: Kullanım amacına göre format seçin
5. **Düzenli raporlayın**: Periyodik raporlar oluşturun
